CSS Box Model

What is it? The box model is the set of properties that define parts of an element that take up space on a web page.

Property Description Notes
width the width of the content area
height the height of the content area
padding the amount of space
border the thickness and style of the border surrounding the content area and padding
margin the amount of space between the border and the outside edge of the element top and bottom margins collapse, meaning they do not add up instead the larger of the two vertical margins sets the distance between adjacent elements
min-width ensures a minimum width of an element's box
min-width ensures a max width of an element's box
overflow controls what happens to content that spills outside of its box
overflow: hidden any content that overflows will be hidden from view
overflow: scroll a scrollbar will be added to the element's box so that the rest of the content can be viewed by scrolling
overflow: visible the overflow content will be displayed outside of the containing element
Resetting defaults * {
margin: 0;
padding: 0;
}		 many deverlopers choose to reset default values so they can truly work with a clean slate. This code resets the default margin and padding. It is often the first rule in an external stylesheet.
visibility elements can be hidden from view with the visibility property
visibility: hidden hides an element
visibility: visible displays an element
visibility: collapse collapses an element
